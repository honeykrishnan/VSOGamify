
# VSOGamify
Gamification in VSO

Pre-requisites:

	Install Node.js

	Have a VSTS account that you can install extensions from marketplace

Clone the code from here

Go to command prompt and the folder and Run

	npm init -y
	
Install the VSS SDK

	npm install vss-web-extension-sdk --save

To create the VSIX 

	tfx extension create
